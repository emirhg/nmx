Plataforma integral de Normas Oficiales Mexicanas (NOMs)

El objetivo de la plataforma es brindar una herramienta electrónica que contenga toda la información publicada en el DOF relacionada a las Normas Oficiales Mexicanas vigentes. Para mantener actualizada la información, la plataforma descarga diariamente el DOF y vincula la información de las NOMs automáticamente, utilizando como referencia la clave de la regulación.

Nuestra labor consistió en descargar más de 8,100 documentos del DOF relacionados directamente a las NOMs desde 1976, para después clasificar y etiquetar cada publicación. Asimismo, vinculamos cada norma a su dependencia, comité, al tipo de bien o servicio regulado, a su rama económica y MIR.

Con base en esta documentación, elaboramos un sitio web que fuera amigable con el usuario y le permitiera consultar el contenido regulatorio de las Normas Oficiales Mexicanas de forma clara y fácil. Los pasos realizados fueron:

  1. Descargamos la información diaria del DOF desde 1910. 

  2. Recolectamos todas las publicaciones referentes a las NOMs a través de un clasificador bayesiano de una librería para procesamiento de lenguaje. 

    1. Utilizamos dos webservices del DOF, los cuales se complementan en casos donde la información no esté completa en alguno de ellos. Sin embargo, uno de ellos tiene caracteres mal codificados para los acentos, por lo que elaboramos una matriz de todas las palabras acentuadas en las NOMs para corregir este error.  

  3. Creamos un registro para cada título dónde se mencione una NOM, si se mencionan varias NOMs creamos un registro por cada clave. 

    1. Un significativo número de NOMs no sigue la nomenclatura convencional, por lo que creamos una base de datos con todas las posibles variaciones. 

  4. Hicimos limpieza manual para obtener una clave NOM estandarizada. 

    1. Hicimos una matriz que corrige los errores tipográficos encontrados en las NOMs. 

    2. Incluimos los distintos cambios de nomenclatura para una misma norma, por ejemplo: NOM-027-SEMARNAT-1996 = NOM-003-RECNAT-1996 = NOM-003-SARH3-1994 

  5. Clasificamos manualmente todas las NOMs vigentes, verificando que se hayan incluido correctamente todos sus documentos. 

  6. Clasificamos manualmente el tipo de documento para los 8,141 registros para crear un clasificador automático (aprendizaje de máquina). 

  7. Considerando que no hay relación directa entre una NOM y su respectiva MIR, revisamos 955 links de COFEMER para ligar cada NOM a su MIR. 

  8. Hicimos una API que genera un URL para consultar la publicación original del DOF. 

  9. Visualizamos y damos funcionalidades de búsqueda en la página de NOMs-IMCO.
